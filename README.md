# SkinScript — Last 11 Board Exams Quiz

A professional UWorld-style quiz application featuring 799 dermatology board exam questions from September 2019 to October 2025, extracted verbatim from the original exam PDFs.

## 🎯 Features

- **799 verified questions** from 11 board exams (Sep 2019 – Oct 2025)
- **UWorld-style interface** — immediate color-coded feedback
- **Three modes**: Tutor (answers after each), Timed (countdown), Review (browse freely)
- **Smart question filtering**:
  - All Questions
  - Unused Only
  - Incorrect Only (review your mistakes)
  - Flagged Only (bookmarked questions)
  - By Topic
- **Progress tracking** with quiz history and performance analytics
- **Start Fresh** option to reset all progress
- **Dark mode** support
- **Offline-capable** — all data stored in localStorage

## 🚀 Getting Started

```bash
npm install
npm run dev
```

Open http://localhost:5173 in your browser.

## 🏗️ Build

```bash
npm run build
```

Output goes to the `dist/` folder.

## 📦 Tech Stack

- React 19
- Vite 8
- Tailwind CSS 4
- Lucide React icons

## 📚 Source

Questions extracted from the original *Last 11 Exams* PDF — preserved **exactly** as written, with no modifications to questions, choices, answers, or explanations.
